# Programs

## LinuxToys

A collection of fun tools and tweaks for the Linux desktop, including games, animations, and visual effects.

**Install via Flatpak:**

```bash
flatpak install flathub io.github.linuxtoys.app
```

---

## NormCap

OCR-powered screen capture tool. Select a region of the screen and it extracts the text from it — great for copying text from images, PDFs, or any non-selectable content.

**Install via Flatpak:**

```bash
flatpak install flathub com.github.dynobo.normcap
```

**Install via pip:**

```bash
pip install normcap
```
