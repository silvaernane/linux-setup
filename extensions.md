# GNOME Extensions

## Copyous

Clipboard manager extension for GNOME Shell. Keeps a history of copied items so you can access previously copied text and snippets.

**Install:** [extensions.gnome.org/extension/copyous](https://extensions.gnome.org/extension/copyous)

---

## Emoji Copy

Quickly search and copy emojis directly from the GNOME Shell top bar. Supports keyboard shortcuts for fast access.

**Install:** [extensions.gnome.org/extension/emoji-copy](https://extensions.gnome.org/extension/emoji-copy)
